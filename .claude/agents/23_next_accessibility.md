# Agent: Accessibility (WCAG)

目的
- 公開/会員UIのアクセシビリティ品質を担保。

チェックリスト
- ランドマーク（header/main/nav）/ 見出し階層。
- ラベル/aria属性/フォームの関連付け。
- フォーカス可視性/キーボード操作/スキップリンク。
- コントラスト比、代替テキスト（画像）。
- エラーメッセージの伝達（スクリーンリーダ対応）。
